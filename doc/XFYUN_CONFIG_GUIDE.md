# 科大讯飞实时语音转写配置指南

## 功能说明
本应用集成了科大讯飞实时语音转写（标准版）接口，支持在语音模式下实时将语音转换为文字。

## 配置步骤

### 1. 注册科大讯飞开发者账号
1. 访问 [科大讯飞开放平台](https://www.xfyun.cn/)
2. 注册账号并登录
3. 进入控制台，创建新应用

### 2. 获取API参数
1. 在应用管理中找到你的应用
2. 获取以下参数：
   - **APPID**：应用ID
   - **API_KEY**：API密钥
   - **API_SECRET**：（实时语音转写大模型接口需要，标准版接口不需要）

### 3. 配置参数
编辑 `src/config/xfyun.ts` 文件，填写你的API参数：

```typescript
export const XFYUN_CONFIG = {
  // 应用ID
  APPID: '你的APPID',
  
  // API密钥
  API_KEY: '你的API_KEY',
  
  // API密钥的密钥
  API_SECRET: '你的API_SECRET',
  
  // 其他配置保持默认即可
};
```

## 使用说明

### 语音转写功能
1. 在"记录梦境"页面切换到"语音模式"
2. 点击麦克风按钮开始录音
3. 录音过程中会实时显示转写的文字
4. 再次点击麦克风按钮停止录音
5. 转写完成后可以编辑文字内容
6. 点击"保存"按钮保存梦境

### 注意事项

1. **网络要求**：需要稳定的网络连接才能使用实时语音转写功能
2. **权限要求**：需要麦克风权限
3. **计费说明**：科大讯飞API需要付费使用，请在控制台查看详细的计费标准
4. **音频格式**：支持16kHz采样率，单声道音频
5. **语言支持**：默认支持中文，可在配置中修改

## 故障排除

### 常见问题

1. **连接失败**
   - 检查网络连接
   - 确认API参数是否正确
   - 检查科大讯飞账号余额是否充足

2. **转写不准确**
   - 确保环境安静，避免背景噪音
   - 说话清晰，语速适中
   - 保持麦克风距离适中

3. **权限问题**
   - 在设备设置中允许应用访问麦克风
   - 重新启动应用

## 技术细节

- **接口类型**：WebSocket实时接口
- **音频格式**：PCM格式，16kHz采样率
- **支持语言**：中文、英文
- **响应时间**：实时转写，延迟约200-500ms
