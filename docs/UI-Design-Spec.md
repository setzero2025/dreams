# 记梦App UI设计规范

**版本**: v1.0  
**日期**: 2026-03-27  
**平台**: iOS / Android  
**设计风格**: 沉浸式深色主题，星空梦境元素

---

## 1. 设计理念

### 1.1 核心设计原则

| 原则 | 说明 |
|------|------|
| **沉浸感** | 深色主题配合星空背景，营造梦境氛围 |
| **简洁性** | 减少视觉干扰，聚焦核心功能 |
| **反馈感** | 操作即时反馈，动画流畅自然 |
| **安全感** | 隐私内容明确标识，数据加密提示 |

### 1.2 视觉风格

- **色调**: 深邃的宇宙蓝 + 梦幻紫 + 星光点缀
- **质感**: 磨砂玻璃效果 + 光晕效果
- **字体**: 无衬线字体，清晰易读
- **图标**: 简约线条风格，配合发光效果

```tsx
<Button
  variant="secondary"
  size="medium"
  onClick={handleClick}
```

- 边框: 2px `#6366f1`

- 圆角: 8px
- 内边距: 12px 24px
- 悬停: 背景变为 `rgba(99, 102, 241, 0.1)`

## 10. 响应式设计

### 10.1 断点设计

| 设备类型 | 屏幕宽度 | 布局调整 |
|----------|----------|----------|
| **手机** | < 640px | 单列布局，简化导航 |
| **平板** | 640px - 1024px | 双列布局，增强内容展示 |
| **桌面** | > 1024px | 多列布局，完整功能展示 |

### 10.2 适配原则

- **文字大小**: 移动端适当放大确保可读性
- **间距调整**: 移动端增加触摸目标大小
- **导航优化**: 移动端使用底部导航栏
- **内容优先级**: 移动端优先展示核心功能

## 11. 可访问性

### 11.1 色彩对比度

- 确保所有文字与背景的对比度 ≥ 4.5:1
- 按钮和交互元素对比度 ≥ 3:1
- 遵循WCAG 2.1 AA级标准

### 11.2 键盘导航

- 支持键盘Tab键导航
- 清晰的焦点状态指示
- 合理的Tab顺序

### 11.3 屏幕阅读器

- 所有交互元素添加无障碍标签
- 图片添加alt文本描述
- 动态内容变化通知屏幕阅读器
