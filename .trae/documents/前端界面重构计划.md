# 前端界面重构计划

## 目标
根据《需求规格说明文档.md》中的功能和交互设计，使用 frontend-design 技能重构现有的 React Native 前端界面。

## 当前状态分析

### 现有页面结构
- `Home.tsx` - 首页（梦境列表、快捷记录）
- `Record.tsx` - 记梦页面（语音/文字记录）
- `DreamDetail.tsx` - 梦境详情
- `Generate.tsx` - 生成中心
- `Profile.tsx` - 个人中心
- `Discover.tsx` - 发现页
- `CreationCenter.tsx` - 创作中心
- `ScriptToVideo.tsx` - 剧本生成视频
- `ScriptViewer.tsx` - 剧本查看
- `ImageViewer.tsx` - 图片查看
- `VideoPlayer.tsx` - 视频播放
- `Login.tsx` / `Register.tsx` - 登录注册
- `Subscription.tsx` - 订阅页面（已有）

### 导航结构
底部 4 栏导航：首页 / 生成 / 发现 / 我的

## 重构计划

### Phase 1: 首页模块优化
1. **Home.tsx 改造**
   - 更新欢迎语文案
   - 添加搜索框组件
   - 实现梦境列表搜索过滤
   - 添加右滑删除功能
   - 优化星空背景动效

2. **DreamDetail.tsx 改造**
   - 完善四个快捷功能按钮（梦生图片/视频/剧情/解读）
   - 添加梦境作品展示区域
   - 添加分享功能（朋友圈/小红书）

### Phase 2: 资产模块新增
1. **新建 Assets.tsx 页面**
   - 顶部 Tab 切换：照片 / 音频 / 视频
   - 照片管理：网格展示、删除、下载
   - 音频管理：列表展示、播放、删除
   - 视频管理：网格展示、播放、收藏、删除、导出

2. **更新导航**
   - 底部导航将"生成"改为"资产"
   - 或添加"资产"入口到"我的"页面

### Phase 3: 我的模块完善
1. **Profile.tsx 改造**
   - 添加统计信息卡片（累计梦境、作品、连续天数）
   - 添加头像更换功能
   - 完善个人信息编辑
   - 添加菜单列表：
     - 我的收藏
     - 心理测评
     - 主题设置
     - 知识库管理
     - 订阅管理
     - 退出登录

2. **新建 PsychologicalTest.tsx 心理测评页面**
   - 时间范围选择（7天/15天/30天）
   - 测评结果展示
   - 改善建议

3. **新建 ThemeSettings.tsx 主题设置页面**
   - 三种模式选择

### Phase 4: 记梦模块增强
1. **Record.tsx 改造**
   - 添加音色记录提示
   - 完善保存后的快捷功能入口

### Phase 5: 生成模块优化
1. **Generate.tsx / CreationCenter.tsx 改造**
   - 完善生成进度展示
   - 添加模型来源标注
   - 完善额度限制提示

### Phase 6: 订阅模块完善
1. **Subscription.tsx 优化**
   - 完善价格展示
   - 添加权益对比
   - 完善支付流程（模拟）

## 设计规范

### 色彩方案
- 主背景：`#0f0f1e`（深邃夜空）
- 卡片背景：`#1a1a2e`（半透明深蓝）
- 主色调：`#6366f1`（梦幻紫）
- 强调色：`#8b5cf6`（渐变紫）
- 文字主色：`#ffffff`
- 文字次要：`rgba(255, 255, 255, 0.7)`

### 动画效果
- 星空背景：浮动星星
- 卡片：悬停/点击缩放
- 按钮：脉动光晕
- 转场：滑动过渡

### 字体
- 标题：粗体 24-28px
- 正文：常规 14-16px
- 标签：中等 12-14px

## 实施顺序

1. **Phase 1** - 首页模块优化（高优先级）
2. **Phase 2** - 资产模块新增（高优先级）
3. **Phase 3** - 我的模块完善（高优先级）
4. **Phase 4** - 记梦模块增强（中优先级）
5. **Phase 5** - 生成模块优化（中优先级）
6. **Phase 6** - 订阅模块完善（中优先级）
