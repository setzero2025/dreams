# 记梦模块设计规格说明书

## Why
当前应用的"首页"和"资产"之间缺少一个专门的"记梦"入口。根据需求规格说明文档，底部导航应该是：首页 / 记梦 / 资产 / 我的。记梦模块需要整合语音记梦（说梦）和文字记梦功能，提供统一的梦境记录入口，并支持保存后直接进行AI创作（梦生图片、视频、剧情、解读）。

## What Changes
- **新增页面**: RecordModule.tsx - 记梦模块主页面
- **修改导航**: 更新 AppNavigator.tsx，将"记梦"作为底部导航的第二个Tab
- **整合功能**: 将原有的 Record.tsx 功能整合到新的记梦模块中
- **新增快捷创作区**: 保存梦境后展示梦生图片/视频/剧情/解读四个快捷入口
- **新增梦境作品预览**: 展示当前梦境已生成的作品

## Impact
- 受影响文件:
  - `frontend/src/navigation/AppNavigator.tsx` - 修改底部导航结构
  - `frontend/src/pages/RecordModule.tsx` - 新增记梦模块主页面
  - `frontend/src/pages/Home.tsx` - 可能需要调整快捷记梦入口
- 用户体验: 提供更清晰的记梦入口，缩短记梦→创作的交互路径

## ADDED Requirements

### Requirement: 记梦模块主页面
记梦模块 SHALL 提供统一的梦境记录入口，整合语音和文字两种记录方式。

#### Scenario: 语音记梦（说梦）
- **WHEN** 用户进入记梦模块
- **AND** 用户选择语音记录方式
- **THEN** 展示录音界面，使用科大讯飞实时语音转写
- **AND** 显示音色记录提示
- **AND** 实时展示转写文字
- **AND** 支持保存录音并提取音色特征

#### Scenario: 文字记梦
- **WHEN** 用户进入记梦模块
- **AND** 用户选择文字记录方式
- **THEN** 展示表单：梦境时间、标题、内容、情绪标签、自定义标签
- **AND** 支持保存梦境到本地/云端

#### Scenario: 快捷AI创作
- **WHEN** 用户成功保存梦境
- **THEN** 展示四个快捷功能按钮：梦生图片、梦生视频、梦境剧情、梦境解读
- **AND** 每个按钮显示当前额度状态
- **AND** 点击后跳转对应创作流程

#### Scenario: 梦境作品展示
- **WHEN** 当前梦境已有生成作品
- **THEN** 在页面底部展示梦境作品列表
- **AND** 支持点击查看详情

## Design Specifications

### 页面结构
```
┌─────────────────────────────────────────┐
│  Header: 记梦                    [关闭]  │
├─────────────────────────────────────────┤
│  Tab切换: [说梦] [写梦]                  │
├─────────────────────────────────────────┤
│                                         │
│  内容区域:                               │
│  - 说梦: 录音按钮 + 转写文本区 + 音色提示  │
│  - 写梦: 表单（时间/标题/内容/情绪/标签）  │
│                                         │
├─────────────────────────────────────────┤
│  [保存梦境] 按钮                         │
├─────────────────────────────────────────┤
│  快捷创作区（保存后显示）:                 │
│  [🖼️梦生图片] [🎬梦生视频]               │
│  [🎥梦境剧情] [🔮梦境解读]               │
├─────────────────────────────────────────┤
│  梦境作品（有作品时显示）:                 │
│  [作品1] [作品2] [作品3] ...             │
└─────────────────────────────────────────┘
```

### 颜色规范
- 背景: `#0f0f1e` (深邃夜空)
- 卡片背景: `rgba(26, 26, 46, 0.8)`
- 主色调: `#6366f1` (梦幻紫)
- 强调色: `#8b5cf6` (渐变紫)
- 文字主色: `#ffffff`
- 文字次要: `rgba(255, 255, 255, 0.7)`

### 动画效果
- Tab切换: 滑动过渡 + 内容淡入
- 录音按钮: 脉冲光晕动画
- 保存成功: 按钮缩放 + 快捷创作区滑入
- 作品卡片: 悬停/点击缩放

### 交互流程
1. 用户点击底部导航"记梦"进入模块
2. 默认展示"说梦"Tab，可切换到"写梦"
3. 用户完成记录后点击"保存梦境"
4. 保存成功后展示快捷创作区
5. 用户可直接点击四个AI创作功能
6. 如有已有作品，底部展示作品列表
